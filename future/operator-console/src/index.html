      Setup guidance

      1. Run make operator-console-check to validate the local console files.
      2. Start the local gateway on loopback, then run make operator-console-serve.
      3. Browser chat relies on /api/chat/stream, but current builds still return buffered pseudo-SSE rather than token-by-token output.
      4. If non-loopback hosting is required, pass --allow-non-loopback and secure access at the host or network layer.
